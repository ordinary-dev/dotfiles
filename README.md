# .files

## Getting started
```sh
git clone git@github.com:ordinary-dev/dotfiles.git ~/.dotfiles
rm -rf ~/.zshrc ~/.config/nvim ~/.config/mpv
ln -s ~/.dotfiles/.zshrc ~/.zshrc
ln -s ~/.dotfiles/neovim/ ~/.config/nvim
ln -s ~/.dotfiles/mpv/ ~/.config/mpv
```

## List of programs
* mpv
* neovim
* zsh
